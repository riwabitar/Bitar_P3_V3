SOLVE FOR X

Participants will solve simple equations individually to figure out which shape to draw. Using a ruler, they will draw each shape to the correct measurement included and place it in order. The main inputs for this exercise are basic math equations, measurements, and placement directions, which guide the drawing process step by step. In the end, the combined shapes will create a recognizable drawing.

Required Tools

  • 8.5 x 11 Grid Paper
  • Pencil
  • Ruler
  • Colored Markers (Red / Blue / Orange / Green)

Number of Participants

1 Participant

Estimated Time

20 Minutes

Instructions

Solving & Drawing

  • Two pieces of paper will be given to you: one with equations and one for drawing.
  • The drawing paper will include the values and color of each shape.
  • Solve each equation for X. The value of X tells you which shape to draw.
  • After solving each equation, you will receive instructions for the measurement and placement of the shape corresponding to the value of X.
  • Use a ruler (for non-round shapes) and a pencil to draw each shape accurately.
  • Draw shapes in the order given so the drawing forms correctly.
  • Continue solving and drawing until the drawing is complete.
  • Trace over the shapes using the colors assigned to each of the shapes.
  • After figuring out what the drawing is, you must add your own design elements to make the drawing complete. You can use the colors of the shapes to draw and/or color in.

if/then

  • If a shape is the wrong size, then erase and redraw it before moving on.
  • If a shape is placed incorrectly, then fix its position before drawing the next shape.
  • If you complete a shape, then trace over it using the assigned color.
